

# Техническое задание для разработчиков API 220volt.kz

Составлю документ с конкретными проблемами и требованиями, который можно передать вашему разработчику.

## Что будет в документе

**Формат**: PDF-документ с техническим заданием

**Содержание**:

1. **Описание проблемы** — параметр `query` не находит товары по полному названию, даже если товар есть в каталоге и в наличии (41 001 шт.)

2. **Воспроизведение бага** — таблица с конкретными запросами и результатами:
   - `query="ECO T8 18Вт 6500К"` → 0 результатов ❌
   - `article="LLE-T8-18-230-65-G13"` → 1 результат ✅
   - Вывод: товар есть, но текстовый поиск его не индексирует

3. **Требования к исправлению**:
   - Параметр `query` должен находить товар по любой комбинации слов из его названия
   - Поиск должен поддерживать числовые характеристики (6500К, 18Вт, 230В)
   - При вводе полного названия товар ДОЛЖЕН присутствовать в результатах

4. **Опциональные улучшения** (nice-to-have):
   - Новый параметр `pagetitle` для точного поиска по названию
   - Поддержка фильтрации по цветовой температуре через `options[]`

## Действие

Сгенерирую PDF-документ в `/mnt/documents/` для передачи разработчику.
